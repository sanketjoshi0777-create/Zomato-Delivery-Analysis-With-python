In this project, I worked on analyzing restaurant data from Zomato using Python. I started by importing and cleaning the dataset to handle missing values and ensure data accuracy. Then, I performed exploratory data analysis (EDA) to understand patterns in ratings, pricing, locations, and cuisines.

I used libraries like Pandas and NumPy for data manipulation and Matplotlib/Seaborn for visualization. I analyzed top cities with the most restaurants, popular cuisines, and the relationship between cost, votes, and ratings. I also created visual charts to present insights clearly.

Overall, I applied data analysis techniques to extract meaningful insights that can help in understanding customer preferences and improving business decisions in the food delivery industry.
